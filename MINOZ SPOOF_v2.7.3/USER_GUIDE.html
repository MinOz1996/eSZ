🚀 การใช้งานครั้งแรก (First Time)
1
เปิดโปรแกรมด้วยสิทธิ์ Admin คลิกขวาที่ START.bat → เลือก Run as Administrator
หน้าต่างจะขึ้นกรอบสีม่วง พร้อม MINOZ Logo สีเขียว
2
ดูค่าเดิมก่อน — กด [4] VIEW กด 4 เพื่อดู ORIGINAL values
แคปหน้าจอเก็บไว้! — จะใช้ตรวจสอบภายหลัง
สถานะจะแสดง [ORIGINAL] ทุกค่า
3
เลือก Mode ที่ต้องการ 1 Ghost Full — แนะนำ, ครบทุก module (100%)
2 Standard — เร็วกว่า, Registry เท่านั้น (70%)
4
[1] Ghost Full — Reboot Prompt จะปรากฏ หลัง spoof เสร็จ โปรแกรมจะถาม:
Restart computer now? [Y/N]
กด Y → รีเครื่องอัตโนมัติใน 5 วินาที
กด N → รีเองทีหลัง (บางค่าอาจยังไม่มีผล)
5
หลัง Reboot — ตรวจสอบ [4] VIEW อีกครั้ง เปิดโปรแกรมใหม่ → กด 4
สถานะกรอบสีม่วงจะแสดง ● Active Spoofing : Ghost Full
ค่าที่เปลี่ยนได้จะแสดง [SUCCESS]
6
เล่นเกม / ใช้งานได้เลย ปิดโปรแกรม เปิดเกม — สถานะจะยังคงอยู่แม้ปิดโปรแกรมไปแล้ว

กรอบสถานะ

🔴 ○ ORIGINAL IDENTITY
ยังไม่ได้ทำ spoof


🟢 ● Active Spoofing : Ghost Full
ใช้ [1] Ghost Protocol แล้ว


🟢 ● Active Spoofing : Standard
ใช้ [2] Standard แล้ว

สถานะใน [4] VIEW

[SUCCESS] — ค่าเปลี่ยนแล้ว active


[PENDING REBOOT] — เขียนแล้ว รอรีเครื่อง


[ORIGINAL] — ค่าจริง ยังไม่ spoof


[UNCHANGED] — user-mode limit

🔒 ระบบ Lock — ป้องกันการ Spoof ซ้อน

หลังจาก Spoof ด้วย 1 หรือ 2 แล้ว:


  • ทั้ง [1] และ [2] จะถูก ล็อค อัตโนมัติ
  • ต้องกด 3 RESTORE ก่อน ถึงจะปลดล็อค
  • ป้องกัน Backup ถูก Overwrite ด้วยค่าปลอม

ถ้าเห็น LOCKED ให้กด [3] Restore → Reboot → Spoof ใหม่
🔄 การ Restore — คืนค่าปกติ
1
เปิดโปรแกรม (Run as Admin) → กด [3] เลือก 3 RESTORE IDENTITY
2
รอ Restore สำเร็จ โปรแกรมจะคืน Registry ทุกค่ากลับเป็นของเดิม
3
Reboot เครื่อง รีเครื่องเพื่อให้ค่าทุกอย่างมีผล
4
ตรวจสอบ [4] VIEW สถานะควรกลับเป็น [ORIGINAL] ทุกค่า
ห้ามกด [1] หรือ [2] ซ้ำก่อน Restore — จะได้ Backup Prompt ให้เลือก SKIP เสมอ
🧹 Deep Clean Traces [5]

ใช้เพื่อลบ Trace ที่อาจถูกตรวจพบ โดยไม่ต้อง Spoof


  • ล้าง Prefetch, Temp files, ShimCache
  • ล้าง USN Journal (NTFS Change Log)
  • ล้าง Event Logs
  • ล้าง MountPoints2

ใช้ [5] ก่อน Spoof ถ้าเคยใช้ Cheat/Tool อื่นมาก่อน
⚠ Backup Warning — เลือก SKIP เสมอ!

ถ้ากด [1] หรือ [2] ขณะที่มี Backup อยู่แล้ว:


โปรแกรมจะถาม [1] SKIP / [2] OVERWRITE / [3] CANCEL
  • [1] SKIP — เก็บ Backup เดิม (ORIGINAL) ไว้ ✓ เลือกอันนี้เสมอ!
  • [2] OVERWRITE — Backup จะเป็นค่าปลอม → Restore จะได้ค่าปลอมกลับมา!
  • [3] CANCEL — ยกเลิก operation
